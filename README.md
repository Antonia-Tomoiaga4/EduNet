# EduNet – Learning Management System

EduNet is a full-stack web application designed to facilitate interaction between students and professors.

Built with Java (Spring Boot) and React (TypeScript), the platform supports course management, assignments, quizzes, and grading, with role-based access control.

##  Features
- Student / Professor roles
- Course creation and management
- Assignment submission and grading
- Quiz creation and evaluation
- RESTful API integration
- PostgreSQL database

##  Tech Stack
- Backend: Java, Spring Boot
- Frontend: React, TypeScript
- Database: PostgreSQL
